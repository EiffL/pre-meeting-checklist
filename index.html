Checklist for the 2020 LSST DESC July Virtual Meeting

Here are a few useful things you can do to prepare yourself for the DESC virtual meeting!

Access meeting resources

Check out and bookmark the meeting agenda page (on all the devices that you plan to use). Make sure you remember your Confluence password.
Check out the logistics page for details on e.g., slack channels, presentation templates, and Zoom access.
If you want to add Sprint ideas, please visit Sprints page to browse and add your hack/sprint ideas! Remember, not all hacks and sprints have to be carried out, so be bold when adding ideas! Also add your name and time constraints at the bottom at the page!
Remind yourself of our Code of Conduct which applies to all participants.
Check out the participant guidelines which list out recommendations for various roles of meeting participants.

Prepare for engaging during the meeting

Ensure that your Zoom and Slack apps are up-to-date.
Add pronouns to your Zoom and Slack accounts e.g., as <First-Name> <Last-Name> (<pronoun>).
Join the following Slack channels:
  • #desc-collab-meeting to hear meeting announcements and to interact with meeting participants.
  • #desc-hack-sprint to hear Sprints-related announcements and to interact with Sprinters.
  • #desc-collab-meeting-hotline to report any technical problems that may arise during the meeting.

Computing resources etc.

Make sure you have a NERSC account and remember your password (nope, a SSH key is not enough) and have 2-step verification (MFA) set up! Need to apply for one? Forget your password? Questions? Ask them in the #desc-nersc Slack channel. Also, make sure you are in the "lsst" group. If not, contact Richard Dubois or Debbie Bard to request to be added to the "lsst" group.
Follow the instructions to install DESC jupyter kernels. You will need them to access DC2 data products or to use DESC software during the Collaboration meeting.
If you have not used any DC2 data tutorial notebooks before, check out the instructions on this step-by-step guide. Also, bookmark the DC2 Data Product Overview page and the DC2-analysis GitHub repo for easy access. Questions? Ask them in the #desc-dc2-users Slack channel.

Social media

If you use social media professionally, please use the #lsstdesc hashtag when reporting highlights from the meeting! Remember the Code of Conduct mentioned above!
